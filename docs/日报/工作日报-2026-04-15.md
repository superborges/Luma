## 技术要点

1. **Tailwind config `borderRadius` 重写**：`rounded-full = 0.75rem = 12px`，非标准的 9999px capsule。这是全局设计语言的关键差异，影响所有按钮/FAB/导航圆角。
2. **AIScore 是 struct**：包含 `.overall: Int` 和 `.scores: PhotoScores`（composition/sharpness/exposure/color/story），非简单 Double。
3. **EditSuggestions 是 struct**：含 crop/filterStyle/adjustments 等结构化字段，非字符串数组。
4. **SwiftUI padding vs CSS padding**：`.padding()` 在 `.frame()` 之外会增大布局尺寸；CSS padding 在元素内部。关键修复是将 `py-8` padding 移到 frame 约束之前。
