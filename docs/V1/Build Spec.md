# Build Spec — Luma V1

## 1. 版本信息

- 产品：Luma（拾光）
- 版本：V1 — 选片体验增强与核心流程打磨
- 日期：2026-04

## 2. 本次版本目标

- 本次版本要解决的核心问题：MVP 已跑通「导入→分组→选片→导出」闭环，但选片效率偏低（AI 评分/废片标签有数据却没在 UI 展示）、照片导入筛选太粗（仅张数）、导出文件命名不可控、快捷键不完整。
- 本次版本最重要的用户任务：让用户在选片时**看到** AI 评分与废片提示，**一键完成**连拍组选优，通过**快捷键**快速决策；从 Mac 照片导入时能**按月份**精准筛选。
- 本次版本完成后，用户能做到什么：
  - 选片效率提升：右栏评分+废片标签辅助判断，快捷键覆盖所有高频操作
  - 照片导入精准化：iCloud 闭环场景下按月份筛选（含照片数/容量预估/重复提醒），而非「最近 N 张」
  - 导出更灵活：文件命名规则可选（保留原名/日期前缀/自定义模板）
  - 分组可调：设置页支持调整时间阈值，适配不同拍摄节奏

## 3. 功能范围

### 本次包含

- **F1 选片右栏增强**：展示本地 AI 评分（五维子分+总分）与废片标签（模糊/过曝等 issues）
- **F2 快捷键补齐**：`U` 回待定、`G` 切网格/单张、连拍组内「采纳 AI 推荐」
- **F3 照片导入筛选**：完全重写 `AppKitPhotosImportPicker`，按月份选择导入（翻页式 UI），每月显示照片数/容量预估，含大月警告与重复导入提醒
- **F4 导出文件命名**：保留原名 / 日期前缀 / 自定义模板（`{date}_{original}`等）
- **F5 设置页增强**：分组时间阈值可调（15/30/60 分钟预设）、缩略图缓存上限、默认命名规则

### 本次不包含

- 云端 AI 评分（多模型/API Key/预算调度）
- AI 修图建议自动生成
- DiskArbitration SD 卡监控迁移
- `~/Downloads` 文件夹监控 / AirDrop 自动导入
- 多语言切换（中/英/跟随系统）
- 导出到云盘（iCloud Drive / Dropbox）

## 4. 用户主路径

1. 用户进入：打开 App → Session 列表
2. 用户看到：可从菜单选择导入来源；若选「Mac·照片 App」，弹出**月份选择器**（按年翻页，每月显示照片数/容量预估/重复提醒）
3. 用户执行：导入完成后进入选片工作区 → 右栏可见 **AI 评分 + 废片标签** → 用快捷键高效决策（P/X/U/G/1-5）
4. 系统反馈：选片自动保存；进入导出时可配置**文件命名规则**
5. 用户完成：导出完成 → 回到列表

## 5. 页面清单

- Session 列表：不变（继承 MVP）
- **选片工作区**：右栏新增 AI 评分卡片+废片标签（F1）；快捷键扩充（F2）
- **月份选择器**：Mac 照片路径弹出按年翻页的月份选择 UI（F3）
- **导出面板**：新增文件命名规则 Section（F4）
- **设置页**：通用 Tab 增加分组阈值/缓存上限/默认命名规则（F5）

## 6. 每页核心任务

### 选片工作区（F1 + F2）

- 页面目标：高效完成照片决策
- 主操作：P 选中 / X 拒绝 / U 回待定 / G 切视图
- 次操作：查看右栏 AI 评分与废片标签；连拍组内采纳推荐

### 月份选择器（F3）

- 页面目标：从 Mac 照片 App 按月份精准筛选一批照片导入
- 主操作：勾选一个或多个月份 → 确认导入
- 次操作：查看每月照片数/容量预估；留意大月警告（>500 张）与重复导入提醒；按年翻页浏览

### 导出面板（F4）

- 页面目标：配置导出选项并执行
- 主操作：选目标 + 命名规则 → 开始导出
- 次操作：选未选中处理方式；查看预览摘要

### 设置页（F5）

- 页面目标：调整全局偏好
- 主操作：调分组时间阈值 / 设缩略图缓存上限 / 选默认命名规则
- 次操作：查看数据目录；打开诊断

## 7. 关键交互规则

- 默认进入时展示什么：选片页右栏默认展示当前选中照片的 EXIF + AI 评分 + 废片标签
- 主按钮点击后发生什么：P/X 操作立即标记并自动跳下一张；U（回待定）标记后不跳转，方便用户重新审视
- 返回逻辑是什么：选片页所有状态自动保存；返回列表无需确认
- 什么时候自动保存：每次决策（Pick/Reject/Pending）立即持久化
- 什么时候要二次确认：月份选择器确认导入前展示已选月份总数与容量预估；删除原图走系统弹窗
- 什么时候给提示：废片标签以半透明叠加 + 角标形式展示；评分低于阈值的照片轻提示

## 8. 状态设计

- 默认状态：右栏评分卡展示总分 + 五维子分 + 文案；无 issues 时标签区域不占位
- 空状态：未评分照片显示「评分中…」或「暂无评分」
- 加载状态：月份选择器在后台加载月份统计数据（`PhotoKitSafetyWrapper` 超时兜底），加载完成后直接展示每月照片数与容量预估
- 成功状态：导入完成进入选片；导出完成显示摘要
- 失败状态：筛选面板权限不足时展示 `PhotosAccessGuidance`；导出失败项可重试
- 异常状态：分组阈值调整后提示「下次导入生效」（不重新分组已有 Session）

## 9. 数据与对象

- 核心对象：`MediaAsset.aiScore` / `MediaAsset.issues`（已有，V1 在 UI 消费）；`PhotosImportPlan`（V1 重写为 `MonthSlot` 月份选择模型）；`ExportOptions` 新增命名规则字段；`GroupingEngine` 时间阈值配置化
- 对象间关系：`AIScore` 五维子分 → 右栏 UI 卡片；`issues` → 废片标签；`PhotosImportPlan.selectedMonths` → 月份选择器
- 用户会修改哪些内容：选片决策、导出选项（含命名规则）、设置偏好（阈值/缓存/命名规则）
- 哪些状态需要持久化：选片决策（manifest）、导出选项（UserDefaults）、设置偏好（UserDefaults）、分组阈值（UserDefaults，下次导入读取）

## 10. 非目标范围

- 这版明确不解决什么：云端 AI 与修图建议、多语言、SD 卡 DiskArbitration、Downloads 监控
- 哪些想法先不做：AI 组名生成（如「清水寺·日落」）、评分校准（20 张参考照）、导出后通知

## 11. 验收标准

- [ ] 选片右栏能看到 AI 总分 + 五维子分 + 废片标签
- [ ] 快捷键 U 回待定 / G 切视图 均可用
- [ ] 连拍组内可一键采纳 AI 推荐
- [ ] Mac·照片导入弹出月份选择器（按年翻页，多月可选）
- [ ] 每月显示照片数与容量预估；含大月警告（>500 张）与重复导入提醒
- [ ] 导出面板可选文件命名规则（≥3 种）
- [ ] 设置页可调分组时间阈值 / 缩略图缓存上限 / 默认命名规则
- [ ] 所有新功能有对应单测或集成测试
- [ ] V1 合约测试 `./scripts/run-v1-contract-tests.sh` 通过
- [ ] 不引入新的重型外部依赖
